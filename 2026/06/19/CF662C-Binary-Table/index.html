CF662C Binary Table

发表于 2026-06-19 阅读次数： 本文字数： 439 阅读时长 ≈ 1 分钟

题意

n×mn\times mn×m 的 010101 数组，可以任意翻转多次行与列，求最少的 111 数。

n≤20,m≤105n\le 20,m\le 10^5n≤20,m≤105

思路

我们发现，如果直接枚举行与列是不行的，但是 n≤20n\le 20n≤20 又提示我们可以枚举行的翻转情况然后考虑列如何翻转。考虑一个列的状态如果是 sss，行的翻转是 xxx，那么新的行状态就是 s⊕xs \oplus xs⊕x（异或）。那么，我们维护一下 sss 的出现次数 aaa，bxb_xbx​ 表示 xxx 中 000 与 111 出现最少的个数，则答案为

∑i=02n−1asbs⊕x\sum_{i=0}^{2^n-1} a_s b_{s\oplus x} i=0∑2n−1​as​bs⊕x​

那么，如果我们考虑枚举 xxx，则上式变成了

fx=∑j⊕k=xajbkf_x=\sum_{j\oplus k=x}a_jb_k fx​=j⊕k=x∑​aj​bk​

变成了标准的 fwt 形式，时间复杂度 O(n2n)\mathcal{O}(n2^n)O(n2n)。
