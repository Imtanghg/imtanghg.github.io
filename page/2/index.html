P10846 [EGOI 2024] Team Coding / 团队编程

发表于 2026-05-01 更新于 2026-06-05 本文字数： 609 阅读时长 ≈ 2 分钟

题意

一棵以 111 为根的树，每个点有颜色。同一层的两个点颜色可以互换。以任意一个点的颜色它的子树经过任意交换的最多跟这个点相同的颜色个数和最小交换次数。

n,∣C∣≤105n,|C|\le 10^5n,∣C∣≤105。

思路

发现第二问是逗你玩的，对于 uuu 和它的最大答案 AAA，第二问答案就是 AAA 减去 uuu 原本和 uuu 颜色相同的个数，这个启发式合并一下就搞完了。

观察部分分发现有颜色数很多和颜色数很少的两种情况，这提醒我们思考根号分治。

对于一个颜色数，如果它有不超过 BBB 个节点。我们可以考虑枚举每一个节点作为负责人，那么我们维护每一层的对应颜色数 cdc_dcd​。那么我们只需要求出负责人在 ddd 层的后代个数 NdN_dNd​。那么对第一问答案的贡献就是 min⁡(cd,Nd)\min(c_d,N_d)min(cd​,Nd​)。后代个数可以通过 dfn 序每一层排序一下二分得到，复杂度 O(nBlog⁡n)O(nB\log n)O(nBlogn)。

对于超过 BBB 个节点的，那么这样的颜色数不超过 nB\frac{n}{B}Bn​ 个。我们考虑去启发式合并一下整棵树全部优先向当前颜色上靠的最大值，那么复杂度就是 O(n2log⁡nB)O(\frac{n^2\log n}{B})O(Bn2logn​)。

我们取 B=nB=\sqrt nB=n​ 平衡复杂度到 O(nnlog⁡n)O(n\sqrt n\log n)O(nn​logn)，但实测跑的飞快。

P3838 [IOI 2017] Toy Train

发表于 2026-05-01 更新于 2026-06-05 本文字数： 527 阅读时长 ≈ 1 分钟

题意

给定一个有向图，玩家 A，B 分别控制一些点。有一些点是特殊点。对于每一个点作为开始位置，A，B 一开始选择将每一个自己的点只保留一条出边，保证形成一条环。A 先手，求是否使得包含一个特殊点。

n≤5000,m≤20000n\le 5000 , m\le 20000n≤5000,m≤20000

思路

首先我们发现，对于一开始作为开始位置可以等价于转化成每次走到一个点选择一条边去走，因为最优情况下这不会改变什么。之后发现，环的长度最大就是 nnn，所以等价于我们只需要去找是否能够经过特殊点无数次。
对于只有一个特殊点的情况，我们考虑有向图博弈求出其他的点先手能否到达这个特殊点的状态 sss。那么对于这个特殊点的归属，如果属于先手控制那么只需要满足存在相邻位置先手能到即可无数次，因为这样就可以形成一个环，即为状态 sss。那么对于后手控制的情况，只要不存在有相邻位置先手到不了，那么就是前面的 sss，否则全 000。

现在思考扩展到多个点。我们发现因为要经过无数个特殊点，那么需要保证一个特殊点每次都可以到达另一个（可以为自己）的特殊点，只要这样就可以走无数次了。所以我们迭代 nnn 次每次删去不能动的特殊点即可。那么 sss 就是最终答案。

时间复杂度 O(nm)O(nm)O(nm)。

P1251 餐巾计划问题

题意

P5101 [JOI 2017 Final] 绳 / Rope

题意

给出 a1,…,ana_1,\ldots,a_na1​,…,an​。每次可以首尾折叠一个偶数回文串，求最终折成长度为 222 且有一个颜色是 c∈{1,…,M}c\in\{1,\ldots,M\}c∈{1,…,M} 的改颜色的最小代价。

思路

考虑一开始染颜色怎么染。发现对于除了首尾以外的奇数连续段，那么它是无法减少长度的。然后需要观察到结论：

能够拼成两个的充要条件是除了首尾两段以外，其它所有连续段长度为偶数。且颜色个数最多为 222。

对于充分性，可以考虑先将最后一段搞成 111。然后对于倒数第二段中线翻折，那么倒数第二段就变成了最后一段，重复即可。

对于必要性，如果对于中间的奇数段，那么它的两端都是异色的，所以永远无法被合并到首尾来消掉。

那么枚举第一段出现位置是首还是第二个位置。考虑一个偶数的连续段可以拆成若干个长度为 222 的段。那么考虑当前的颜色 ccc。则如果两个颜色都是 ccc 那么直接过，否则如果一个是 ccc 则将另一个也改成 ccc 即可。否则求其它剩余颜色个数最多的是哪个即可。

那么，我们假设对于颜色 ccc 总共出现了 NcN_cNc​ 次，设当前长为 222 的段两个颜色 c1,c2c_1,c_2c1​,c2​ 总共出现了 fc1,c2f_{c_1,c_2}fc1​,c2​​ 次，则相当于要求：

n−Nc+min⁡i≠c(fi,c−Ni)n-N_c+\min_{i\neq c}(f_{i,c}-N_i) n−Nc​+i=cmin​(fi,c​−Ni​)

=n−Nc−max⁡i≠c(−fi,c+Ni)=n-N_c-\max_{i\neq c}(-f_{i,c}+N_i) =n−Nc​−i=cmax​(−fi,c​+Ni​)

那么可以认为，对于所有段中有 ccc 的点，假定为 (c,p)(c,p)(c,p) 且 p≠ip\neq ip=i，我们认为将 NpN_pNp​ 减少 111 然后维护一下 max⁡i≠c(Ni)\max_{i\neq c}(N_i)maxi=c​(Ni​) 即可。
